# Blackjack Game

This is an implementation of the Blackjack card game using HTML, CSS, and JavaScript. 

## How to Play

1. Start by entering your initial money amount.
2. Click on the "Deal" button to start a new round.
3. Enter the amount you want to wager for the current round.
4. The dealer and player will be dealt two cards each.
5. Click on the "Hit" button to request another card.
6. Click on the "Stand" button to end your turn.
7. The dealer will then play their turn.
8. The results will be displayed, indicating whether you win, lose, or tie.
9. Your balance will be updated accordingly based on the outcome.
10. Continue playing rounds until you decide to end the game.

## Features

- Simple and intuitive user interface.
- Ability to place wagers and track your balance.
- Dealer logic for playing their turn.
- Win, loss, and tie outcomes.
- Option to end the game and view your final balance.

## Setup

1. Clone the repository or download the source code.
2. Open the `index.html` file in a web browser.
3. Start playing the game by following the on-screen instructions.
